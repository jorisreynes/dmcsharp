La version compilée du projet se trouve dans dmcsharp/bin/Debug/net7.0/app.dll

Le projet a été développé avec VS Code

Pour lancer le projet, à la racine faire un dotnet run

Certains parties sont manquantes, notamment les spécificités de chaque classe, j'ai manqué de temps à la fin, j'ai préféré avoir quelque chose qui fonctionne, il y a le système de duel, qui fonctionne avec n'importe quel nombre de personnages (battle royale), les classes sont implémentées avec leurs attributs (vie, attaque, etc), il est possible de choisir sa classe au début, comme demandé il faut appuyer sur une touche pour passer au round suivant, et il y a un classement à la fin,
